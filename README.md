# Higher Standards Software Design

Marketing site for the Higher Standards software design studio.

## Pages

| Path | Description |
|------|-------------|
| `/` | Studio home |
| `/work` | Project index |
| `/work/[slug]` | Individual case studies |
| `/philosophy` | Design philosophy |

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Works on Vercel or Render as a standard Next.js app:

```bash
npm run build
npm start
```
